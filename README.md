# iam-frontend

## Before installation

Install Python
https://www.python.org/downloads/release/python-2710/

On Windows
Put python2 as path variable

Use Node version 14

## Project setup
```
npm install @vue/cli
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
